# Фінальний проект для першого модуля Python University від JavaRush

## Телеграм бот зі зверненням до ChatGPT

### Режими роботи

1. *"Випадковий факт"*
За командою /random ChatGPT видає цікаві факти

2. *"ChatGPT інтерфейс"*
Дозволяє вільне спілкування з ChatGPT у форматі запитання-відповідь. Без збереження контексту

3. *"Діалог з відомою особистістю"*
Дозволяє обрати зі списка певну історичну особу і поспілкуватися з її віртуальний аватаром, згенерованим ChatGPT

4. *"Квіз"*
За обраними темами генеруються питання і підраховується кількість прапвильних відповідей із загальної кількості

5. *"Перекладач"*
Користувач обирає мову і отримує переклади введених фраз на цю мову

6 *"Рекомендації щодо фільмів та книг"*
Користувач може обрати категорію з переліку фільми, книги, музика, додати жанр і отримувати рекомендації для ознайомлення.
На наступну рекомендацію впливає реакція на попередню - можна відмічати ті, що не подобаються
